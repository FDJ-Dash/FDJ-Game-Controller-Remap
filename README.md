# Game-Controller-Remap
## This is an app aimed towards game controller devices not recognized by some games but still recognized by the OS.

- Here you get to choose if you want to deactivate Controller Remap, and the kind of Mode to use. The real flexibility happens thanks to the Ini file variables.

## How does it work?
- The App will detect your device and remap its commands to a keyboard equivalent execution which is read from the GameControllerRemap.ini file.
- You get to choose the game mode, and adapt to a kind of camera rotation that the game allows.
- You can also modify the ini file to match your specific game needs.
